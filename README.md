# 体育用品销售系统

## 项目简介
体育用品销售系统是专门用来销售体育运动相关商品。 技术上使用Java语言、SSH框架，Mysql数据库技术、JSP页面。 分两个子系统：前台系统和后台管理系统。前台系统是给消费者浏览商品、加入购物车、下单等操作。后台管理系统管理员登录后可以对商品、用户、订单进行新增、编辑、删除操作。适合作为毕业设计、大作业、课程设计等
